# Legacy Navigator

Legacy Navigator is a full-stack platform designed to guide families through afterlife administration by simplifying complex legal and administrative processes through automation and AI-assisted support.

## Project Overview

Legacy Navigator helps users manage posthumous administrative tasks by:
- Automating asset discovery  
- Using OCR to extract and structure a Schedule of Assets  
- Generating court-ready document packages  
- Providing assistance through an agentic AI chatbot for uncertain legal queries  

The platform aims to reduce complexity, errors, and emotional burden by offering a clear and guided digital experience.

## Tech Stack

- Frontend: React, TypeScript, Vite  
- UI: Tailwind CSS, shadcn-ui  
- Backend & AI: Python, Agentic AI  
- Other: OCR integration, Full-stack architecture
